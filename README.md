# Build #

```
docker-compose build
```

# Run #

```
docker-compose up -d --force-recreate --remove-orphans
```

# Test locally #

```
http://localhost
```

# Host via AWS S3 #

```
http://osps-scratch.s3-website-ap-southeast-2.amazonaws.com/
```

# Logs #

```
docker-compose logs -f
```

# Processes #

```
docker-compose ps
```

# Shutdown #

```
docker-compose down
```
